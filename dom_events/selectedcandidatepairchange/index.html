selectedcandidatepairchange

A selectedcandidatepairchange event occurs when the ICE agent selects a new pair of candidates which together describe the endpoints of a viable connection. The pair of candidates is described by an RTCIceCandidatePair object which contains one RTCIceCandidate representing the local end of the connection, and another representing the remote end of the connection.

Together, the candidates can be used to establish a connection to be used by the RTCIceTransport, and, by extension, by an RTCPeerConnection.
