Ember.HashLocation (protected)

Extends: Ember.Object
Defined in: packages/ember-routing/lib/location/hash_location.js:15
Module: ember

Ember.HashLocation implements the location API using the browser's hash. At present, it relies on a hashchange event existing in the browser.

Using HashLocation results in URLs with a # (hash sign) separating the server side URL portion of the URL from the portion that is used by Ember.

Example:

App.Router.map(function() {
  this.route('posts', function() {
    this.route('new');
  });
});

App.Router.reopen({
  location: 'hash'
});

This will result in a posts.new url of /#/posts/new.
