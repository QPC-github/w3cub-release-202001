tensorflow::ops::Atan2

#include <math_ops.h>

Computes arctangent of y/x element-wise, respecting signs of the arguments.

Summary

This is the angle ( [-, ] ) such that [ x = r () ] and [ y = r () ] where (r = (x^2 + y^2) ).

Arguments:

  • scope: A Scope object

Returns:

  • Output: The z tensor.
