Node.js libraries

Drivers and extensions

  • rethinkdbdash by @neumino
    An alternative Node.js driver with native promises and a connection pool.

  • rethinkdb-co by @hden
    Allows using ECMAScript 6 generators with RethinkDB callbacks.

  • rdb-cursor-stream by @guillaumervls
    Replaces cursors with streams.

  • connect-rethinkdb by @guillaumervls
    A RethinkDB session store for Connect, similar to connect-redis.

  • express-session-rethinkdb by @armenfilipetyan
    Use RethinkDB as session store with Express 4.x framework. Adapted from connect-rethinkdb.

  • Rethinkdb-pool by @hden
    Connection pool for RethinkDB connections.

  • second-thought by @robconery
    A light abstraction layer over RethinkDB adding methods you “wished you had.”

  • rethinkdb-fixtures by @athlite
    Easily load fixtures into RethinkDB for testing purposes.

ORMs

  • js-data-rethinkdb by @jmdobry
    A RethinkDB adapter for js-data, a database-agnostic ORM for Node.js and the browser.

  • Thinky by @neumino
    JavaScript ORM for RethinkDB.

  • ThinkAgain by @mbroadst
    A fork of Thinky with first-class support for JSON Schema.

  • JugglingDB-RethinkDB by @fuwaneko
    A RethinkDB adapter for JugglingDB, a multi-database ORM for Node.js.

  • Osmos by @mtabini
    A store-agnostic object data mapper for Node.js with support for RethinkDB.

Integrations

  • rabbitMQ
    Listen for changes in a RethinkDB table over RabbitMQ.

  • koa-rethinkdb by @hden
    Koa middleware that automatically manages connections via a connection pool.

  • thinky-loader by @mwielbut
    A general-purpose model loader for the Thinky ORM. (Supersedes sails-hook-thinky.)

  • hapi-rethinkdb-crud by @athlite
    Basic CRUD mapping between hapi and RethinkDB.

Python libraries

ORMs

  • rwrapper by @dparlevliet
    An ORM designed to emulate the most common usages of Django’s database abstraction.

  • pyRethinkORM by @JoshAshby
    A Python ORM for RethinkDB.

  • rethink by @caoimhghin
    A Python object mapper in the style of App Engine NDB.

  • remodel by @linkyndy
    A simple but powerful and extensible Object Document Mapper for RethinkDB. (Also see the pip package.)

Integrations

  • rabbitMQ
    Listen for changes in a RethinkDB table over RabbitMQ.

  • celery-backends-rethinkdb by @pilwon
    Celery’s custom result backend for RethinkDB.

  • flask-rethinkdb by @linkyndy
    A Flask extension that adds RethinkDB support (also see the pip package).

Ruby libraries

ORMs

  • NoBrainer by @nviennot
    A Ruby ORM designed for RethinkDB. (Read our quickstart tutorial.)

Integrations

  • Epiphy by @kureikain
    A light persistence framework for RethinkDB.

  • rabbitMQ
    Listen for changes in a RethinkDB table over RabbitMQ.

  • lotus-rethinkdb by @angeloashmore
    A RethinkDB adapter for Lotus::Model, a persistence framework for Ruby.
