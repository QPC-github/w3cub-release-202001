panos_lic – apply authcode to a device/instance

  • Synopsis
  • Requirements
  • Parameters
  • Examples

  • Status

DEPRECATED

Removed in Ansible:
version: 2.12
Why: Consolidating code base.
Alternative: Use https://galaxy.ansible.com/PaloAltoNetworks/paloaltonetworks instead.

Synopsis

  • Apply an authcode to a device.
  • The authcode should have been previously registered on the Palo Alto Networks support portal.
  • The device should have Internet access.

Requirements

The below requirements are needed on the host that executes this module.

  • pan-python

Parameters

Parameter Choices/Defaults Comments
auth_code
- / required
authcode to be applied
force
boolean
    Choices:
  • no ←
  • yes
whether to apply authcode even if device is already licensed
ip_address
string / required
IP address (or hostname) of PAN-OS device.
password
string / required
Password for authentication.
username
string
Default:
"admin"
Username for authentication.

Examples

- hosts: localhost
  connection: local
  tasks:
    - name: fetch license
      panos_lic:
        ip_address: "192.168.1.1"
        password: "paloalto"
        auth_code: "IBADCODE"
      register: result
- name: Display serialnumber (if already registered)
  debug:
    var: "{{result.serialnumber}}"

Return Values
